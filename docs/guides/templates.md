## What are Templates?

Templates are pre-structured documents that help you quickly create common types of notes with consistent formatting.

## Using Templates

### Create from Template

1. Navigate to the Templates page
2. Browse available templates
3. Click on the template you want
4. A new document is created with the template structure
5. Fill in the details

### Customize Templates

You can modify any template:

1. Open a document created from a template
2. Adjust the structure as needed
3. Save your changes
4. The original template remains unchanged

## Creating Custom Templates

### Method 1: From Existing Document

1. Create a document with your desired structure
2. Format it as a template
3. Save it in your workspace
4. Use it as a reference for future documents

### Method 2: Template File (Coming Soon)

## Template Best Practices

1. **Keep it flexible**: Don't over-structure templates
2. **Use placeholders**: Mark areas to fill in (e.g., `[Your Name]`)
3. **Include instructions**: Add comments for guidance
4. **Test templates**: Try them out before sharing
5. **Version templates**: Update them as your needs evolve

## Common Use Cases

### Work Templates

- **Weekly Report**: Track weekly progress and goals
- **1-on-1 Meeting**: Structure manager/employee meetings
- **Sprint Planning**: Agile sprint documentation
- **Code Review**: Technical review checklist

### Personal Templates

- **Morning Routine**: Daily planning and reflection
- **Book Notes**: Capture reading insights
- **Recipe**: Cooking instructions and variations
- **Travel Plan**: Itinerary and packing lists

### Educational Templates

- **Class Notes**: Lecture documentation
- **Study Guide**: Exam preparation
- **Lab Report**: Scientific experiment documentation
- **Essay Outline**: Academic writing structure

## Sharing Templates

### Export Template

1. Open your template document
2. Click **"Export"**
3. Choose Markdown format
4. Share the file with others

### Import Template

1. Receive a template file
2. Copy the content
3. Create a new document
4. Paste the template structure

## Troubleshooting

### Template Not Appearing

1. Refresh the Templates page
2. Check if templates are loaded
3. Clear browser cache if needed

### Formatting Lost

1. Use markdown for consistent formatting
2. Avoid complex HTML in templates
3. Test templates after creation
