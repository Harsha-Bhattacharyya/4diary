# Keyboard Shortcuts Guide

Master 4Diary with these keyboard shortcuts for maximum productivity.

## Quick Reference

### Essential Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+Q` / `Cmd+Q` | Open/Close Quick Note |
| `Ctrl+Shift+F` / `Cmd+Shift+F` | Open Search |
| `Ctrl+Shift+C` / `Cmd+Shift+C` | Open Calculator |
| `Ctrl+S` / `Cmd+S` | Save current document |
| `Escape` | Close modal or exit mode |

### Editor Shortcuts

| Shortcut | Action |
|----------|--------|

## Global Shortcuts

These work anywhere in 4Diary:

### Quick Note
**`Ctrl+Q` (Windows/Linux) | `Cmd+Q` (Mac)**

- Opens Quick Note modal instantly
- Press again to close (content preserved)
- Works on any page in 4Diary

**Usage Example**:
```
1. Working in document editor
2. Press Ctrl+Q
3. Type quick thought
4. Press Escape to return
5. Continue editing
```

### Save Document
**`Ctrl+S` (Windows/Linux) | `Cmd+S` (Mac)**

- Manually triggers document save
- Usually auto-save handles this
- Useful for forcing save before closing
- Shows "Saving..." indicator

### Close/Cancel
**`Escape`**

- Closes Quick Note modal
- Exits Reader Mode
- Closes emoji picker
- Cancels current operation
- Dismisses dialogs

## Editor Shortcuts

### Text Formatting

#### Bold
**`Ctrl+B` (Windows/Linux) | `Cmd+B` (Mac)**

- Makes selected text **bold**
- Or toggle bold on/off
- Works on partial selections

**Usage**:
```
1. Select text
2. Press Ctrl+B
3. Text becomes bold
4. Press Ctrl+B again to unbold
```

#### Italic
**`Ctrl+I` (Windows/Linux) | `Cmd+I` (Mac)**

- Makes selected text *italic*
- Toggle italic formatting
- Combine with bold for ***bold italic***

#### Underline
**`Ctrl+U` (Windows/Linux) | `Cmd+U` (Mac)**

- Underlines selected text
- Toggle underline formatting
- Works with other formatting

#### Strikethrough
**`Ctrl+Shift+X` (Windows/Linux) | `Cmd+Shift+X` (Mac)** *(Planned)*

- Adds ~~strikethrough~~ to text
- Currently use markdown: `~~text~~`

### Search Notes

#### Open Search
**`Ctrl+Shift+F` (Windows/Linux) | `Cmd+Shift+F` (Mac)**

- Opens powerful search modal
- Search across all documents
- Use filters for precise results
- Navigate with arrow keys

**Search Workflow**:
```
1. Press Ctrl+Shift+F
2. Type search query
3. Use ↑↓ to navigate results
4. Press Enter to open document
5. Press Escape to close
```

**Search Filters**:
- All - Search everywhere
- Title - Search only titles
- Content - Search document content
- Tags - Search tags
- Folder - Search folders

### Calculator

#### Open Calculator
**`Ctrl+Shift+C` (Windows/Linux) | `Cmd+Shift+C` (Mac)**

- Opens built-in calculator
- Supports LaTeX math expressions
- View calculation history
- Insert results into editor

**Calculator Workflow**:
```
1. Press Ctrl+Shift+C
2. Enter expression (e.g., sqrt(16))
3. Press Enter to calculate
4. Click "Insert to Editor" (optional)
5. Press Escape to close
```

**Supported Operations**:
- Basic: +, -, *, /, ^
- Functions: sqrt(), sin(), cos(), tan(), log()
- Constants: pi, e

### Links

#### Remove Link
**`Ctrl+Shift+K` (Windows/Linux) | `Cmd+Shift+K` (Mac)** *(Planned)*

- Removes link from text
- Keeps text, removes URL
- Currently: Edit link and clear URL

### Undo/Redo

#### Undo
**`Ctrl+Z` (Windows/Linux) | `Cmd+Z` (Mac)**

- Undo last action
- Multiple levels of undo
- Works for text edits and formatting

#### Redo
**`Ctrl+Shift+Z` (Windows/Linux) | `Cmd+Shift+Z` (Mac)**  
**Alternative**: `Ctrl+Y` (Windows/Linux)

- Redo previously undone action
- Multiple levels of redo
- Returns to state before undo

### Block Commands

#### Slash Commands
**`/`** (forward slash)

Opens block menu with options:
- `/h1` - Heading 1
- `/h2` - Heading 2
- `/h3` - Heading 3
- `/bullet` - Bullet list
- `/number` - Numbered list
- `/check` - Checklist
- `/code` - Code block
- `/quote` - Quote block

**Quick Workflow**:
```
1. Type / at start of line
2. Menu appears
3. Type command or use arrows
4. Press Enter to insert
```

## Markdown Shortcuts

Type these patterns for instant formatting:

### Headings

| Type | Result |
|------|--------|
| `# ` | Heading 1 (largest) |
| `## ` | Heading 2 |
| `### ` | Heading 3 |
| `#### ` | Heading 4 |
| `##### ` | Heading 5 |
| `###### ` | Heading 6 (smallest) |

*Note*: Space after # is required

### Lists

| Type | Result |
|------|--------|
| `- ` or `* ` | Bullet list |
| `1. ` | Numbered list |
| `- [ ] ` | Checklist (unchecked) |
| `- [x] ` | Checklist (checked) |

**Auto-continuation**:
- Press Enter in list → new list item
- Press Enter on empty item → exit list

### Text Formatting

| Type | Result |
|------|--------|
| `**text**` | **Bold text** |
| `*text*` or `_text_` | *Italic text* |
| `~~text~~` | ~~Strikethrough~~ |
| `` `code` `` | `Inline code` |

### Quotes and Code

| Type | Result |
|------|--------|
| `> ` | Quote block |
| ` ``` ` | Code block |
| ` ```js ` | Code block with language (JavaScript) |

### Horizontal Rule

| Type | Result |
|------|--------|
| `---` | Horizontal line/divider |
| `***` | Horizontal line/divider |

## Platform-Specific Notes

### Windows/Linux

- Most shortcuts use `Ctrl` key
- `Alt` key for secondary shortcuts
- Right-click for context menus

### macOS

- Most shortcuts use `Cmd` (⌘) key
- `Option` (⌥) for secondary shortcuts
- Some shortcuts differ (e.g., Cmd+Q for Quick Note)

### Chromebook

- Use `Ctrl` key like Windows/Linux
- Some shortcuts may conflict with Chrome OS
- Use Search key as Alt equivalent

## Tips for Efficiency

### 1. Learn Gradually

Don't memorize all at once:
- Week 1: Master Quick Note (`Ctrl+Q`)
- Week 2: Add formatting (`Ctrl+B`, `Ctrl+I`)
- Week 3: Learn navigation (`Ctrl+F`)
- Week 4: Master slash commands (`/`)

### 2. Muscle Memory

Build habits:
- Use shortcuts instead of clicking
- Practice daily shortcuts
- Avoid mouse when possible
- Speed increases over time

### 3. Context Matters

Different shortcuts for different modes:
- Editor: Formatting shortcuts
- Kanban: Board navigation
- Reader Mode: Reading shortcuts
- Quick Note: Fast capture

### 4. Combine Shortcuts

Chain shortcuts for power moves:
```
1. Ctrl+Q (open Quick Note)
2. Type content
3. Ctrl+S (save to workspace)
4. Escape (close)
```

### 5. Custom Workflows

Create personal shortcut workflows:

**Writing Workflow**:
```
1. Ctrl+Q for idea
2. Save with Ctrl+S
3. Open document
4. Ctrl+/ for block commands
5. Ctrl+B, Ctrl+I for formatting
```

## Troubleshooting

### Shortcut Not Working

**Possible Causes**:
1. Browser extension conflict
2. OS-level shortcut override
3. Keyboard layout difference
4. Focus not in editor

**Solutions**:
1. Disable conflicting extensions
2. Check OS shortcut settings
3. Try alternative shortcut
4. Click in editor before shortcut

### Conflicts with Browser

Some shortcuts conflict with browser:

| Conflict | 4Diary | Browser | Solution |
|----------|---------|---------|----------|
| `Ctrl+W` | N/A | Close tab | Use carefully |
| `Ctrl+T` | N/A | New tab | No conflict |
| `Ctrl+Q` | Quick Note | Quit (macOS) | Override or use button |
| `F5` | N/A | Refresh | No conflict |
