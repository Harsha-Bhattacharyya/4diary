## 8. Responsive Design

### Desktop (1920x1080)
- Full toolbar with all buttons visible
- Wide editor area (max-width: 1024px)
- Floating pen button: bottom-right corner
- Side-by-side statistics in 2x2 grid

### Tablet (768x1024)
- Toolbar buttons remain visible
- Editor width adapts
- Statistics in 2x2 grid
- All features accessible

### Mobile (375x667)
- Toolbar scrollable horizontally
- Full-width editor
- Floating pen button scaled appropriately
- Statistics stack vertically if needed
- Touch-optimized button sizes (44x44px minimum)
