# Security Policy for 4diary
# 
# 4diary is a privacy-focused, end-to-end encrypted note-taking application.
# We take security seriously and appreciate responsible disclosure of security vulnerabilities.
#
# If you discover a security vulnerability, please:
# 1. Do NOT open a public GitHub issue
# 2. Report it via GitHub Security Advisories (preferred)
# 3. Or email us at security@4diary.app
#
# We aim to:
# - Acknowledge receipt within 48 hours
# - Provide an initial assessment within 7 days
# - Keep you informed of our progress
# - Credit you in our security acknowledgments (if desired)
#
# Thank you for helping keep 4diary secure!
